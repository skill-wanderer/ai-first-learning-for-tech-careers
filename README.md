# AI-First Learning for Tech Careers

This repository stores the learning material for the course:

https://dojo.skill-wanderer.com/courses/ai-first-learning-for-tech-careers

## Course Overview

This course teaches a system to learn anything through AI-guided problem solving.

The structure is intentionally scaffolded first so the flow, sequence, and lesson boundaries are stable before full lesson content is published.

Learners will eventually practice how to:

- start with assignments instead of lectures
- use AI as a thinking partner
- break down ambiguous problems into smaller tasks
- reflect and extract reusable knowledge
- build a repeatable AI-powered learning system for career growth
